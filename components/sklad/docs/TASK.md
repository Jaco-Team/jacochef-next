## 1. Объединение модулей

Нужно объединить в один новый модуль:

- `recept_module_new_2` — полуфабрикаты и рецепты
- `site_items_new` — новые товары
- `ed_izmer` — единицы измерения

Уточнение scope:

- `sklad_items_module_new` как текущий FE-модуль не идет в объединение
- это самостоятельный модуль, пока что
- и в текущем scope нового `sklad_items` не делаем отдельный tab/screen copy для товаров склада
- `Item` остается upstream source entity для составов и связей, но не отдельным разделом нового модуля на этом этапе

Текущее указание:

- новые модули не переиспользуем
- код из них копируем
- в новый модуль вставляем отдельно

Отдельно:

- у recipes / полуфабрикатов должна быть общая category model
- внутри показывать, сколько рецептов / заготовок используется в этой категории

Follow-up по рецептам и полуфабрикатам:

- рецепты надо сделать категорией полуфабрикатов
- у рецептов не будет своей отдельной категории
- то есть для production-family должна быть одна category model:
  - `Полуфабрикаты / Рецепты`
  - внутри нее рецепты и полуфабрикаты живут в одном category tree / one category set

Reference screenshot:

- [recipe-pf-category-reference.png](/home/ted/JACO/git/jacochef-next/components/sklad/docs/assets/recipe-pf-category-reference.png)

Важно:

- этот скрин показывает старый FE-экран как reference контекста, а не target layout
- новый UI не должен копировать эту старую компоновку один в один
- из этого follow-up надо брать business rule:
  - recipe category is not a separate business entity from semi-finished category

Также:

- перенести права доступа на новый модуль

## 6. Удаление позиций

Во все указанные модули склада нужно добавить кнопку `Удалить`.

Серверная логика:

- проверять, что позиция прямо сейчас нигде не участвует
- проверять, что позиция никогда не участвовала
- только тогда разрешать удаление
- если где-то используется или использовалась, показывать, что удаление запрещено

Варианты реализации:

- проверка при удалении
- либо вычислять флаг при открытии позиции

Также:

- на удаление нужны отдельные права доступа
